>new
    Create new matrix

>help
    Guide

>save
    Saves matrix to a file

>matrix
    Output matrix

>rtm
    Shapes the matrix into a form of rtm

>det
    Calculates determinant of the matrix

>T
    Transports the matrix 

>*matrix
    Multiplies current amtrix with input matrix

>+matrix
    Adds input matrix to a current amtrix

>* k i
    Multiply line i by k
    Ri <- k * Ri

>/ k i
    Devide line i by k
    Ri <- Ri / k

>+ i j 
    To line i add line j with mult. k
    Ri <- Ri + k * Rj

>- i j 
    From line i subtract line j with mult. k
    Ri <- Ri - k * Rj

>swap i j
    Swap line i and j
    Ri <-> Rj

>krat 
    Get current krat

>undo
    Undo change

>revert
    Opposite of undo

>change
    Change one line

>chelem
    Change one element

>n* k
    Find inverted element(*) for k with current krat

>n+ k
    Find inverted element(+) for k with current krat

>setk k
    Set krat to k
